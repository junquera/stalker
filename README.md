## Install

In `requeriments.txt`:

```
git+https://gitlab.com/junquera/stalker.git#egg=stalker
```

## Usage

```python
import stalker

# Define which elements we desire
# for example Tor URLs
s = stalker.Stalker(tor=True)

elements = s.parse(input_text)
```
